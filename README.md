# moonshot

Created for GitHub Game Off 2020.

You are the moon! Collect stars while avoiding meteors. Can you get a high score?

## tools used

* [libGDX](https://libgdx.badlogicgames.com/)
* [Tiled](https://www.mapeditor.org/)
* Android Studio
* Inkscape
* Paint.net
* git & GitHub
